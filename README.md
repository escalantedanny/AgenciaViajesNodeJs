# AgenciaViajesNodeJs
pagina Agencia de viajes con NodeJs, servidor Express y bases de datos con ORM Sequalize 
Javascript Avanzado

para Correr y crear la aplicacion debemos ejecutar el siguiente comando

sudo npm install

como segundo paso debemos ejecutar la query sql en nuestro servidor local
el archivo se encuentra en 
Server - Config - agenciadeviajes.sql

al finalizar ya configurado debemos correr nuestro servidor con 

npm run start y asi levantaremos nuestra aplicacion
